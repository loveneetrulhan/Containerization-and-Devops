#  Containerization and DevOps Lab Experiments

##  Overview

This repository contains all the lab experiments performed as part of the **Containerization and DevOps course**. The experiments focus on understanding Docker, containers, images, volumes, environment variables, networking, CI/CD, and orchestration tools.

---

##  Objectives

* To understand the concept of containers and virtual machines
* To install and use Docker
* To create and manage Docker images and containers
* To learn Dockerfile creation
* To work with volumes and environment variables
* To implement Docker Compose for multi-container applications
* To understand CI/CD pipelines using Jenkins
* To perform static code analysis using SonarQube
